<p align="center">XiaoMi LAN/cloud control CLI for Node.js</p>

## Installation

```shell
pnpm add -g @homeiot/xiaomi-cli
```

## Usage

### Help

```shell
miot --help
```

### Discover LAN devices

```shell
miot discover
```

### Login XiaoMi Cloud

```shell
miot login
```

### Get XiaoMi Cloud devices

```shell
miot
```
